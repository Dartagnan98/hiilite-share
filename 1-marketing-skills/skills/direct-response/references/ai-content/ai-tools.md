## Tools You Need for AI Content

### Text Generation
- **Claude** -- best for long-form, nuanced content. Projects feature for persistent context
- **ChatGPT** -- custom GPTs for recurring tasks. Good general purpose
- **Poppy** -- specialized for email sequences and marketing copy

### Video Generation
- **VEO 3 (Google Flow)** -- text-to-video generation for ad hooks
- **AI Avatar platforms** -- synthetic spokesperson videos
- VEO 3 fast model: 20 credits per generation

### Voice & Audio
- **ElevenLabs** -- voice changing, voice cloning, text-to-speech
- Used to improve voice quality from AI video generations
- Voice changer feature: drag in video, select voice, generate improved audio

### Image Generation
- **Midjourney** -- highest quality AI images, character consistency features
- **DALL-E** -- integrated with ChatGPT
- **Flux** -- open-source alternative

### Video Editing
- **CapCut** -- free editing for assembling AI-generated clips
- Extract audio, add captions, speed up b-roll, crop, resize
- One-click auto-captions feature

### Presentation
- **Gamma** -- AI-powered presentations for offers and pitches

### Automation
- **AirTable** -- database for tracking topics, scripts, and content pipeline
- **Chrome extensions** -- custom tools for saving content ideas
- **n8n** -- workflow automation for connecting AI tools

## Creating Internal Apps with Claude

### What You Can Build
- Custom script writers that produce consistent content
- Ad copy generators loaded with your winning examples
- Email sequence writers trained on your brand voice
- Content research assistants with your niche knowledge
- Client report generators
- SOPs and process documentation tools

### How to Set Up Claude Projects
1. Create a new project in Claude
2. Write detailed instructions (the mega prompt)
3. Upload reference files:
   - Winning examples (scripts, ads, emails)
   - Brand voice guidelines
   - Word blacklists
   - Offer details and audience info
4. Upload additional context documents
5. Test with sample inputs
6. Refine instructions based on output quality
7. Share with team members

### Custom GPTs in ChatGPT
- Similar concept: persistent instructions + uploaded files
- Custom GPTs can be shared or published
- Good for team-wide tools
- Conversation starters for common use cases

## The VEO 3 Rapid Hook System

### Overview
Generate 50 scroll-stopping video hooks using AI in 1-2 hours.

### Step 1: Generate 50 Hook Ideas (Claude)
1. Open Claude prompt #1 (19-page hook generation framework)
2. Paste in your sales page copy
3. Paste in VSL transcript
4. Paste in best-performing ad transcripts
5. Hit enter -- Claude generates 50 hooks in ~30 seconds

### Step 2: Convert Hooks to VEO 3 Prompts (Claude)
1. In same chat, paste Claude prompt #2 (29-page VEO 3 prompt framework)
2. Claude converts all 50 hooks into video production specs with JSON prompts
3. Each hook gets multiple clip prompts (face-to-camera + b-roll)

### Step 3: Generate Video Clips (Google Flow VEO 3)
1. Open Google Flow, create new project
2. Set to text-to-video, VEO 3 fast model
3. Copy-paste each prompt from Claude
4. Generate clips for each hook (face-to-camera + multiple b-roll clips)

### Step 4: Improve Audio (ElevenLabs)
1. Download video from VEO 3
2. Upload to ElevenLabs voice changer
3. Select a high-quality voice
4. Generate improved speech
5. Download the cleaned audio

### Step 5: Assemble in CapCut
1. Drag in video clip + improved audio
2. Extract and delete original audio
3. Create compound clip (merge video + new audio)
4. Trim dead space
5. Set aspect ratio (1:1 for feed, 9:16 for stories/reels)
6. Speed up b-roll clips for engagement
7. Crop out any text errors from AI generations
8. Add auto-captions, style them, split into readable chunks
9. Export as individual hook files

### Step 6: Create Full Ads
1. Import hook video + proven ad body
2. Stack hook at the beginning
3. Export as complete ad
4. Repeat with different hooks on the same ad body
5. Result: 50 new creative variations from one ad body

## Key Principles

1. AI tools are a production pipeline, not a magic button
2. Input quality determines output quality (garbage in, garbage out)
3. Build persistent tools (projects/GPTs) for recurring tasks
4. VEO 3 + ElevenLabs + CapCut = complete video ad production pipeline
5. 50 hook variations in 1-2 hours is possible with the right system
6. Always improve AI voice with ElevenLabs (raw AI voice quality is poor)
7. Speed up b-roll clips for more engaging videos
8. Auto-captions are essential -- split into readable chunks
9. Test AI-generated hooks against proven manual hooks
10. The system scales -- same process works for any offer, any niche
