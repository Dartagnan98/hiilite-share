## Quick Prompts vs Mega Prompts

### Quick Prompts
- Short, one-sentence instructions
- Good for fast, rough outputs
- "Write me an email about X"
- Fine for brainstorming, not for production content

### Mega Prompts
- Detailed, multi-page instructions with examples
- Produce consistent, high-quality outputs every time
- Include: constraints, context, examples, formatting details
- Required for any marketing-grade content

---

## The 5-Point Prompting Framework

### 1. Desired Output
- Clearly state what you expect in the response
- Be specific about tone, content, format
- "Write a LinkedIn post" vs "Write a LinkedIn post that sounds like a busy CEO sharing a hard-learned lesson, in conversational tone"

### 2. Constraints
- Set limits: max word count, subject boundaries, what NOT to include
- "Do not start until the user provides a topic"
- "Keep under 200 words"
- "Do not use these words: revolutionize, delve, navigate, leverage"

### 3. Context
- Give the AI a backdrop: who you are, your audience, the situation
- "We are a fashion brand that sells undergarments"
- "Our audience is business owners making $50K-$500K/year"
- The more context, the better the output

### 4. Examples (MOST IMPORTANT)
- The #1 factor in output quality
- Give 3-5 examples of what you want the output to look like
- For scripts: provide actual winning scripts as reference
- For emails: provide emails in the tone/style you want
- For ads: provide high-performing ad copy
- Load up as many examples as possible

### 5. Formatting Details
- Specify structure: subheadings, bullet points, numbered lists
- Define sections: hook, explainer, CTA
- Tell it the exact output format you need

## Using Claude Projects for Content Creation

### What Claude Projects Are
- Custom AI environments with pre-loaded instructions and reference files
- Instructions persist across all conversations in the project
- Upload reference documents, examples, and guidelines
- Create once, use daily

### Setting Up a Script Writer Project
1. Write detailed prompt instructions (the 5-point framework)
2. Upload example scripts (50-70 winning examples)
3. Upload "never use these words" lists
4. Upload brand voice guidelines
5. Upload offer details and audience info
6. Start conversations by providing topics

### Example: Viral Hook Script Writer
**Loaded with:**
- 67 example hooks with structure breakdowns
- Instructions for hook/explainer/CTA format
- Brand voice constraints
- Word blacklists
- Retention mechanism requirements (but/therefore storytelling)
- Shareability elements (cliffhangers, surprise endings)

### Results
- Process that took 1 hour/day now takes minutes
- Team members can use the same project
- Consistent quality across all content
- Automated with AirTable for topic tracking

## Creating Ads with AI

### Process
1. Load ChatGPT/Claude with your offer details, audience info, and winning ad examples
2. Provide brand voice guidelines and constraints
3. Request specific ad formats (carousel, single image, video script)
4. Generate multiple variations
5. Edit and refine the best ones
6. Test in campaigns

### Tips
- Always provide winning ad examples as reference
- Specify the platform (Facebook ad copy differs from Google ad copy)
- Include your USP, social proof stats, and offer details in the context
- Generate 10-20 variations and pick the best 3-5 to test

## Creating Emails with AI

### Hammer Them Email Sequences
- Use Poppy (AI email writer) or Claude with email examples
- Load up your best-performing email sequences
- Provide the sequence structure (6 emails per day)
- Context: what the lead did (booked a call, downloaded something, watched VSL)
- Generate value-dense emails that build trust and handle objections

### Short-Form Scripts with AI
- Load examples of your best short-form video scripts
- Provide topic/hook ideas
- Generate 5-10 script variations per topic
- Edit for voice and accuracy
- Use for TikTok, Reels, Shorts content

## Presenting Offers with Gamma

- Use Gamma (AI presentation tool) to create visual offer breakdowns
- Input offer details, AI generates branded presentation
- Good for pitch decks, client proposals, webinar slides
- Faster than building from scratch in PowerPoint/Canva

## Key Principles

1. The more specific your prompt, the better the output
2. Examples are the #1 input that drives quality
3. Build persistent projects/GPTs for recurring content types
4. Create "never use" word lists to avoid AI-sounding copy
5. Load 50+ examples for best results
6. Quick prompts for brainstorming, mega prompts for production
7. AI is a force multiplier, not a replacement for judgment
8. Always edit AI output -- never publish raw
9. Test AI-generated copy against human-written copy
10. Build once, use daily -- projects save hours
