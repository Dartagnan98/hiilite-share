# ManyChat Playbook

Source: Jeremy Haynes / Megalodon Marketing - Master Internet Marketing (Week 5, Lessons 16-25)

---

## Overview

ManyChat is a Messenger/Instagram DM automation platform for lead generation, nurture, and sales. Combines chatbot automation with human live chat to move prospects through your funnel via conversational marketing.

---

## Core Components

### 1. Audience
- Your subscriber base within ManyChat
- Subscribers added through: growth tools, ads with Messenger destination, manual opt-in, keyword triggers
- Segment audience by: behavior, tags, custom fields, engagement level
- Key principle: build a Messenger list just like you build an email list

### 2. Automation
- The core engine of ManyChat
- Triggered sequences and flows that run without manual input
- Types of automations:
  - Welcome messages (first interaction)
  - Keyword-triggered responses
  - Follow-up sequences after opt-in
  - Abandoned cart recovery
  - Event reminders
  - Re-engagement campaigns

### 3. Flows
- Visual conversation builders (flowchart-style)
- Each flow = a conversation path with branches based on user responses
- Elements within flows:
  - Text messages
  - Images and galleries
  - Quick reply buttons
  - User input collection
  - Conditional logic (if/then branching)
  - Delays and timing
  - Actions (add tags, set custom fields, notify team)
- Use flows for: lead qualification, product recommendations, FAQ handling, appointment booking

### 4. Sequences
- Time-based message series (like email drip campaigns but in Messenger)
- Schedule messages over days/weeks
- Use for: nurture sequences, launch sequences, educational content delivery
- Key difference from flows: sequences are time-based, flows are conversation-based

### 5. Keywords
- Trigger automations when users type specific words
- Set up keyword triggers for:
  - Product inquiries ("pricing", "cost", "how much")
  - Support requests ("help", "support", "problem")
  - Content delivery ("free guide", "download", "checklist")
  - Campaign-specific triggers (Instagram comment automation: "DM me LINK")
- Can be exact match or contains match
- Layer keywords to catch variations

### 6. Broadcasts
- One-time messages sent to your subscriber list (like email blasts)
- Types:
  - Promotional broadcasts (within 24-hour engagement window)
  - Non-promotional broadcasts (for re-engagement)
  - Sponsored messages (paid, goes to anyone)
- Rules:
  - Must be within 24-hour window of last interaction (or use sponsored messages)
  - Don't over-broadcast (people will unsubscribe/block)
  - Segment broadcasts by interest/behavior for relevance

### 7. Growth Tools
- Methods to grow your ManyChat subscriber list:
  - Facebook comment automation (comment-to-DM)
  - Instagram comment automation (most powerful current growth tool)
  - Website widgets and popups
  - Landing page opt-ins
  - QR codes
  - Ref URLs (shareable links)
  - Customer chat plugin on website
- Instagram comment automation workflow:
  1. Post content with a CTA: "Comment [KEYWORD] to get the free guide"
  2. ManyChat detects the keyword
  3. Auto-sends DM with the resource + follow-up flow
  4. User is now a subscriber

### 8. Live Chat
- Human takeover when bot can't handle the conversation
- Use for: complex questions, high-ticket sales conversations, customer support
- Setup:
  - Train bot to recognize when to hand off to human
  - Notification system for team when live chat needed
  - Tag conversations for follow-up
  - Set business hours for live chat availability
- Best practice: bot handles 80% of conversations, humans handle 20% (high-value interactions)

### 9. Settings
- Bot behavior configuration
- Key settings:
  - Default reply (what happens when bot doesn't understand)
  - Welcome message (first-time subscriber greeting)
  - Ice breakers (suggested conversation starters)
  - Main menu (persistent menu options)
  - Notification preferences
  - Team member access and roles
  - Integration connections (CRM, email, Zapier)

## Strategy: ManyChat for Lead Generation

### Comment-to-DM Funnel (Instagram)
1. Create valuable content post
2. CTA: "Comment [WORD] to get the free [thing]"
3. ManyChat auto-DMs with the resource
4. Follow-up flow qualifies the lead
5. If qualified -> book a call or send to sales page

### Webinar Registration via ManyChat
1. Ad or post drives to Messenger
2. Bot delivers value + webinar pitch
3. Collects name, email, phone via conversation
4. Registers them and sends confirmation
5. Reminder sequence in Messenger (higher open rates than email)

### E-commerce Product Finder
1. User clicks "Shop" or triggers keyword
2. Bot asks preference questions
3. Recommends products based on answers
4. Links directly to product pages
5. Abandoned cart follow-up if they don't purchase

## ManyChat Best Practices

### Conversation Design
- Keep messages short (2-3 sentences max per bubble)
- Use quick reply buttons for easy responses
- Add personality -- conversational, not robotic
- Include images and GIFs where appropriate
- Always give users a way out ("Not interested? No worries!")
- Test flows with real users before going live

### Compliance
- Always get consent before messaging
- Include opt-out option in every sequence
- Follow Facebook/Instagram messaging policies
- Don't spam -- respect the 24-hour window rule
- Non-promotional content only outside engagement window

### Integration
- Connect to your CRM (GoHighLevel, HubSpot, etc.)
- Sync with email platform for multi-channel sequences
- Use Zapier for custom integrations
- Push qualified leads to sales team in real-time
- Track attribution: which growth tool -> which conversion

### Metrics to Track
- Subscriber growth rate
- Open rates (typically 80%+ for Messenger)
- Click-through rates
- Conversion rates by flow
- Unsubscribe/block rate
- Live chat volume and resolution rate
- Revenue attributed to ManyChat conversations

---

## Key Advantages Over Email

- Open rates: 80%+ vs. 20-25% for email
- Response rates: 30-40% vs. 2-3% for email
- Speed: instant delivery and response
- Interactivity: buttons, carousels, quizzes
- Personalization: conversational and dynamic
- Mobile-native: where people already spend time

## Limitations

- Platform dependency (Facebook/Instagram can change rules)
- 24-hour messaging window for promotional content
- Less suited for long-form content delivery
- Requires ongoing maintenance and monitoring
- Users can block you more easily than email unsubscribe
