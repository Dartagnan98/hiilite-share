Source: MIM Week 6 - Page Building (Lessons 9-18)

## Overview

This guide covers the building blocks for creating high-converting landing pages, funnels, and websites using page builders (ClickFunnels, GoHighLevel, WordPress, etc.). From Jeremy Haynes' MIM Week 6 curriculum.

---

## The 23 Building Blocks

Every landing page or funnel page is built from a combination of these core element types. Master these and you can build any page.

---

## The Builder Basics

### How Page Builders Work
- Drag-and-drop interface
- Sections (rows) contain columns
- Columns contain elements
- Elements are the individual building blocks
- Mobile and desktop views need separate optimization
- Always preview on mobile before publishing

### Section/Row Structure
- Full-width or contained-width sections
- Background colors, images, or videos on sections
- Padding and margins control spacing
- Responsive behavior (how sections stack on mobile)

## Element Types

### 1. Media Elements
- **Images** -- product photos, lifestyle images, screenshots, logos
- **Videos** -- VSLs, testimonials, explainer videos, background videos
- **Image carousels** -- multiple images in a slideshow
- **Background video** -- auto-playing video behind content
- Best practices: compress images for speed, use WebP format, lazy-load below fold

### 2. Text Elements
- **Headlines** -- H1, H2, H3 hierarchy for SEO and readability
- **Subheadlines** -- support the main headline with detail
- **Body text / paragraphs** -- longer explanatory content
- **Bullet points** -- feature lists, benefits, included items
- **Styled text blocks** -- callouts, quotes, highlighted sections
- Best practices: keep headlines under 10 words, use benefit-driven language, left-align body text

### 3. Form Elements
- **Input fields** -- name, email, phone, custom fields
- **Dropdown selects** -- multiple choice options
- **Checkboxes** -- terms acceptance, opt-in consent
- **Submit buttons** -- the primary conversion action
- **Multi-step forms** -- break long forms into digestible steps
- Best practices: fewer fields = higher conversion, use placeholder text, make buttons contrast with page color

### 4. Advanced Form Elements
- **Calendar/scheduling embeds** -- Calendly, ScheduleOnce integration
- **Application forms** -- qualifying questions with conditional logic
- **Quiz/survey elements** -- interactive qualification tools
- **File upload fields** -- for applications requiring documents
- **Phone number with country code** -- international formatting
- Best practices: use conditional logic to show/hide fields, validate inputs in real-time

### 5. Countdown Timers
- **Fixed deadline timers** -- counts down to a specific date/time
- **Evergreen timers** -- unique countdown per visitor (cookie-based)
- **Session timers** -- countdown that resets on return visit
- Creates urgency and drives action
- Best practices: only use if the deadline is real, pair with scarcity messaging, place near CTA

### 6. Content Blocks
- **FAQ accordions** -- expandable question/answer sections
- **Testimonial blocks** -- styled customer reviews with photos
- **Pricing tables** -- side-by-side plan comparisons
- **Feature grids** -- icon + text blocks for features/benefits
- **Trust bars** -- logos of clients, press mentions, certifications
- **Progress bars** -- show completion percentage (for multi-step flows)
- **Tab sections** -- content organized by clickable tabs
- **Social proof notifications** -- "John from NYC just purchased" popups

## Landing Page Structure (Recommended Order)

### Above the Fold
1. **Hero headline** -- primary value proposition
2. **Subheadline** -- supporting detail
3. **Hero image or video** -- visual proof/context
4. **Primary CTA button** -- the action you want them to take
5. **Trust indicators** -- star ratings, client count, certifications

### Below the Fold
6. **Social proof section** -- testimonials, case studies, logos
7. **Problem/agitation** -- describe the pain they're feeling
8. **Solution** -- how your offer solves the problem
9. **Features/benefits** -- what's included (use bullet points)
10. **More testimonials** -- different formats (video, text, screenshots)
11. **Pricing** -- clear, simple pricing with CTA
12. **FAQ section** -- address top 8-10 objections
13. **Final CTA** -- repeat the primary call to action
14. **Guarantee** -- risk reversal (money-back, satisfaction guarantee)

## CRO Principles for Page Building

### Split Testing
- Test one element at a time
- Headlines have the biggest impact on conversion
- Button color/text matters less than button copy/placement
- Always test mobile separately from desktop

### Call Funnels
- Landing page drives to a booked call
- Pre-qualification via form before booking
- Thank you page with videos + next steps

### Low Ticket to High Ticket
- Use a low-cost front-end offer to acquire customers
- Upsell to higher-ticket offers post-purchase
- Tripwire + core offer + premium upsell

### Book Funnels
- Free + shipping book offer as lead magnet
- Captures buyer data at low cost
- Upsell sequence follows the book purchase

## Key Principles

1. Mobile-first design -- majority of traffic is mobile
2. Speed matters -- every second of load time costs conversions
3. One page, one goal -- don't give multiple CTAs that compete
4. Headlines do the heavy lifting -- test them relentlessly
5. Social proof is non-negotiable -- testimonials, logos, numbers
6. FAQ sections handle objections without requiring a phone call
7. Countdown timers work but only with real deadlines
8. Fewer form fields = higher conversion rates
9. Video above the fold increases time on page
10. Every section should earn its place -- if it doesn't move toward conversion, remove it
